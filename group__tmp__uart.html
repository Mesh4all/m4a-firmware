Hardware

This section shows, which pin are used to connect with UART device. in this case to hc-06 to share the UART with bluetooth connections.

Function GPIOs Remarks
UART_DEV(1)_TXD GPIO21 Serial transmitter data port
UART_DEV(1)_RXD GPIO22 Serial receiver data port

Basics

When you are running this example, open your terminal and put "send message" until your shell. Follow the instructions AT to configure you bluetooth in your m4a-wrover board and start the communications.

Flashing

If you want to upload the example to your board, needs to compile first, additional to this you could initialize the terminal and watch the results

make -C examples/tmp_uart flash term

Uart example

this is an uart example for this example i used esp32-wrover-kit and one bluetooth device hc-06

to configure the bluetooth device write in the shell:

1) AT for inter at mode 2) AT+RESET for reset the device 3) AT+PIN password for change password 4) AT+NAME name for change name

to send messages through uart to the bluetooth device write send message since the shell

to compile this example:

make -C examples/tpm_uart BOARD=your_board PORT=/dev/your_port flash term
