◆ set_wifi_mode()

esp_err_t set_wifi_mode ( )

this function set the wifi mode and execute the connection, it find the wifi mode in the nvs and it verify if already saved the mode in the nvs.

If the nvs return error or the wifi_mode is equal to WIFI_MODE_NULL will take the defaults params for give warranty to the user that wifi init correctly this is very important because if something failed the wifi will stop.
