Detailed Description

This is the storage module where you can save data in the flash memory. This data should not be greater than 64 bytes. Data formats allowed are string and uint8.

This function is executed to save any data type or any strutc, The function saves all the information compressed in EEPROM/FLASH_MEMORY.

The data is stored in each page of the EEPROM/MEMORY_FLASH section.
