method to encode data

Parameters
[in]keyMake a name to an element of map.
[in]valueEncode a value uint8_t or uint8* that belongs to [key named element]
[out]cbor_codedBuffer of coded message in cbor, \ this contains all elements that will be saved
[out]lenbuffer size.

Format of element to be Encoded: { "key_named" : 1523 }; TO understand could see these groups of bytes as string.

{ "key_name" : "a5f/s*"}

Note
: Take careful to don't send any char or string under uint8_t and uint8_t* types this function could be take like an array of bytes.
