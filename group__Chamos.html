Detailed Description

CHAMOS

The module CHAMOS (Communication Handler for Addressing Management | Origin Server) It's used for server-side communication for nib routing table. This module also works with the Chamoc - Client tool . this will let you generate a MSG_NIB_ADD or MSG_NIB_DEL request via local host (Linux) and start an intercommunication between a pc and the m4a-24g board.

This module contains:

  • Turning on UDP socket.
  • Receive the message.
  • Process the message and adding it to NIB.
  • Send the ack and the nack message to the client.

Frame:

  • First number is the type which is given by:
MSG_NIB_ADD = 2,
MSG_NIB_DEL = 3,
  • The second value is the sequence number seqno.
  • The third value is the buffer size.
  • From the fourth value is the ipv6 address.
