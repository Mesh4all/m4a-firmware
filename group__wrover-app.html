This is an app of esp32-wrover using a ILI_Screen and Wifi_STA configuration to get values of Bitcoin and Ethereum.

M4a-wrover-app

This example contains an interaction of a udp server on a m4a-wrover board. Sending requests and getting responses to know the values of two cryptocurrencies (Bitcoin and Ethereum).
